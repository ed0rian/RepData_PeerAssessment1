Imputing missing values

Define a function 'avgSteps' that returns the average number of steps from table 'dailyActivity', for a given interval 'x'.

avgSteps <- function(x) { return(dailyActivity[interval == x]$avgSteps) }

missingRows <- which(is.na(activityData$steps))

Are there differences in activity patterns between weekdays and weekends?

Create the new factor variable 'dayType' which indicates whether a given date is a weekday or a weekend day (Saturday or Sunday).

cleanActivityData[, 
                  dayType := ifelse(is.element(weekdays(cleanActivityData[, date]), 
                                               c('Saturday', 'Sunday')
                                              ), 
                                    'Weekend', 
                                    'Weekday'
                             )
                 ]
cleanActivityData[, dayType := as.factor(dayType)]

Aggregate the mean number of steps per 5 minute interval and dayType into the variable 'avgSteps'. Same as above, convert the interval into an extra 'time' variable. Since there is no time in R without date, the current date is used as date for that variable. Keep the (now redundant) 'interval' variable for now.

cleanDailyActivity <- cleanActivityData[, 
                              list(avgSteps = mean(steps)), 
                              by = list(interval, dayType)
                              ]
cleanDailyActivity[, 
                   time := as.POSIXct(sprintf("%05.2f", 
                                              cleanDailyActivity$interval/100
                                             ), 
                                      format = "%H.%M"
                           )
              ]
